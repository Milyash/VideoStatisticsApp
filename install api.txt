I Install npm
II Install MongoDb

1. fetch feature/api-node branch from github
2. Go into root folder
3. Create a package.json file : npm init
4. Install the framework Express : npm install express --save
5. Install the dependencies of Epxress:
	the ORM Mongoose : npm install mongoose
	body-parser (used to parse post request) : npm install body-parser
	a file parser (used to load local config) : npm install fs
6. Set your local variable in the file src/Api/config_local.json
7. Run the server : node app.js
8. Test :
	send post data from the dummy pages (cf index.html)
	get the data to localhost/api/videos
